# Open Prompting

Открытая база знаний о том, как использовать искусственные интеллекты в работе. Мы помогаем освоить ИИ дизайнерам, редакторам и руководителям, чтобы они не тратили время на рутину. Для этого экспериментируем с нейросетями, а результаты выкладываем в двух форматах: рецепты и приёмы.

## Рецепты

Рецепты — пошаговые инструкции, которые решают типичные рутинные задачи.
