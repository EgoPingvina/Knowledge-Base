# Написать черновик контент-плана

Допустим, вы готовите контент-план для корпоративного блога в новой для вас области. Обычно для этого выясняют, о чём в блогах пишут конкуренты. Нейросети быстрее человека анализируют большие объёмы информации. Попросите ЧатГПТ сделать черновик контент-плана по вашему заданию.

Если вы находитесь в России, почитайте [о работе ЧатГПТ в разных регионах](https://github.com/Open-Prompting/Knowledge-Base/blob/main/content/articles/ruchatgpt/).

## Как сделать
1. Описать своего клиента: особенность его бизнеса, целевую аудиторию.
2. Описать контент-план: как часто и как долго должны выходить статьи.
3. Собрать всё в промт:
```
Сделай контент-план корпоративного блога на 12 месяцев. 
Компания — производит мебель из дерева по индивидуальному заказу.
Особенность — премиальные материалы.
```

## Что может пойти не так
ЧатГПТ может генерировать слишком общие или слишком частные темы. Если вас не устраивает большая часть предложенных тем, переформулируйте промт. Или попросите ЧатГПТ сгенерировать больше вариантов и выберите из них наиболее подходящие вам.
